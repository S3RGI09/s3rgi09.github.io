Web en otras darknets

Puedes acceder a la version de esta web en la red I2P y en la red Freenet. Tambien tengo una tienda de scripts en la red Tor

Ambas webs (DeepWeb, I2P, Freenet y Tor) estan protegidas con la mejor estructura web y seguridad posibles. En la red I2P, el servidor web esta gestionado por la organizacion I2P, en Freenet, no hay webs como tal, si no que son todo archivos,en la red Tor, la web esta protegida por CloudFlare, y en la deepweb (donde estas ahora mismo) el servidor es gestionado por GitHub y protegido por CloudFlare.
